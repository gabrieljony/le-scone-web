## Stacks

Esse projeto foi construido com Angular 10.0.1, com a finalizade de usar as tecnicas do [Sass](https://sass-lang.com/):

- Variáveis
- Nesting
- Mixins
- Extend
- Operadores
- Condicionais
- Funções
- Loop
